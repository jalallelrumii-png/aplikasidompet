Deploy Guide
Deploy ke Netlify dari HP
GRATIS • 5 MENIT
Ikuti langkah di bawah untuk deploy app ini ke internet agar bisa diinstall sebagai APK native di semua HP!
1
Download file app ini
Kamu butuh 3 file: index.html, sw.js, dan manifest.json. Sudah ada di folder download kamu dari Claude.
2
Buka Netlify di browser
Buka Chrome → ketik app.netlify.com → Login atau daftar gratis dengan email.
Buka Netlify →
3
Add new site → Deploy manually
Klik "Add new site" → pilih "Deploy manually" → akan muncul area upload.
4
Upload FOLDER (bukan zip!)
⚠️ Penting: Netlify harus menerima folder, bukan zip. Di HP Android, tap area upload → pilih "Upload folder" → pilih folder dompetku yang berisi 3 file. Jika tidak ada opsi folder, gunakan Chrome desktop mode.
5
Dapat URL → Install!
Netlify kasih URL seperti https://dompetku-xyz.netlify.app. Buka URL itu di Chrome → tunggu banner "Install" → klik → App terinstall di homescreen! ✅
?
Alternatif: GitHub Pages
Upload index.html, sw.js, manifest.json ke GitHub repo → Settings → Pages → Deploy from main branch → dapat URL → install!
